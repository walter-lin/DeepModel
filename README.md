## 在本機執行

Google提供的colab，除了支援iPython的格式(就是jupyter所使用的那種)，還有免費提供一定程度的GPU。如果需求增加可再升級成付費版本。

1. 請資料整個打包至Google雲端硬碟 

2. 利用colab開啟train_model_cnn.ipynb(注意附檔名)
